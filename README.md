# fizz-buzz

Determines if an array contains numbers divisibly by 3 or 5.

Installation
------------

```bash
npm install fizz-buzz
```

Usage
-------

```js
var fizzbuzz = require('fizz-buzz');

fizzbuzz('Amor, Roma') // true
fizzbuzz('race car') // true
fizzbuzz('10001') // true
fizzbuzz('100100') // false

```
