# fizz-buzz

Determines if an array contains numbers divisibly by 3 or 5.

Installation
------------

```bash
npm install --save fizz-buzz
```

Usage
-------

```js
var fizzbuzz = require('fizz-buzz');

fizzbuzz([1,2,3,4,5]) // [ 1, 2, 'fizz', 4, 'buzz' ]
fizzbuzz([10,11,12,13,14,15]) // [ 'buzz', 11, 'fizz', 13, 14, 'fizz' ]

```
